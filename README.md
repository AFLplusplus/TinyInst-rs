# tinyinst-rs

FFI to [TinyInst](https://github.com/googleprojectzero/TinyInst). Created for [LibAFL](https://github.com/AFLplusplus/LibAFL).

## Running the test

1. Run `cargo make build_test` to build the test binary
2. Run `cargo test` to run the test
